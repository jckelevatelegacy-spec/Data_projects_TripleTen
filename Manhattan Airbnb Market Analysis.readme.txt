Manhattan Airbnb Market Analysis (Excel Project)

The goal of this project was to analyze the Manhattan vacation rental market using Airbnb data to identify the most attractive neighborhoods and property types for investment. The analysis focused on rental demand, property size preferences, and estimated revenue potential, with results presented through pivot tables and visualizations.

The Data

The dataset was provided as part of a guided project and included multiple related tables:

Listings Sheet: each row corresponds to an individual Airbnb listing

id: unique identifier for each listing
neighborhood: name of the neighborhood where the listing is located
bedrooms: number of bedrooms in the listing
number_of_reviews_ltm: number of reviews in the last 12 months (used as a proxy for demand)
Additional listing attributes used for filtering and segmentation

Calendar Sheet: each row represents a daily availability record for a listing

listing_id: unique identifier linking to listings
date: specific calendar date
available: whether the listing was available or rented (f indicating rented)
adjusted_price: nightly price of the listing

To support analysis, additional calculated columns were created, including cleaned fields and revenue metrics.

The Process

I began by cleaning and preparing the dataset to ensure accuracy and consistency. This included:

Standardizing neighborhood names by removing inconsistent capitalization and trailing spaces, stored in a new column neighborhood_clean
Handling missing bedroom values by converting blanks into studio apartments (0 bedrooms) using an IF function, stored in bedrooms_clean

Next, I used pivot tables to analyze demand and identify the most attractive opportunities:

Ranked neighborhoods based on number_of_reviews_ltm to determine the top 10 most attractive areas for vacation rentals
Analyzed property size distribution to identify the most popular listing types (studios, 1-bedroom, 2-bedroom)
Compared property size preferences across neighborhoods to determine optimal investment strategies

To estimate revenue potential, I engineered new metrics:

Created a revenue_earned column in the calendar dataset, assigning nightly revenue when listings were rented and $0 otherwise
Used a SUMIF() function to aggregate 30-day revenue per listing into the listings dataset
Filtered listings using a top_listing flag based on ideal neighborhood and property size combinations

Finally, I built pivot tables and visualizations to rank listings by revenue and highlight top-performing investment opportunities.

Results

The analysis identified Harlem, Hell’s Kitchen, and the Upper West Side as some of the most attractive neighborhoods based on rental activity. Across nearly all top neighborhoods, 1-bedroom units emerged as the most in-demand property type, with studios being more popular in specific areas like Midtown.

Revenue analysis revealed significant earning potential among top listings, with the highest-performing property generating approximately $29,940 in a 30-day period, projecting to substantial annual revenue when scaled .

By combining demand (reviews) with revenue estimates, the analysis provided a clear, data-driven framework for identifying high-performing investment opportunities.

Conclusion

This project strengthened my ability to use Excel for end-to-end data analysis, including data cleaning, feature creation, pivot table analysis, and revenue modeling. I also developed experience translating raw data into actionable business recommendations.

The final deliverable includes structured analysis, documented assumptions, and visualizations designed to support investment decision-making.
